The Global Biodiversity Information Facility (GBIF) is an intergovernmental network and data infrastructure providing open access to biodiversity data spanning the globe and the tree of life. GBIF harvests the entire OBIS network — so many MBON datasets are discoverable here as well — and assigns citable DOIs to both datasets and downloads, exposing a powerful API and occurrence download service.

Because GBIF mirrors records without OBIS's marine-specific QA/QC and measurement-or-fact extensions, the MBON Data catalog prefers the OBIS version where a dataset appears in both. GBIF remains invaluable for cross-taxon discovery and for datasets in the OBIS network that have not yet been ingested into OBIS.
