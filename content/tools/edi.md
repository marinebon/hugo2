The Environmental Data Initiative (EDI) is an NSF-funded repository that preserves and publishes environmental and ecological data, notably from the Long Term Ecological Research (LTER) network. Several MBON programs — such as the Santa Barbara Channel MBON — archive their nearshore kelp-forest and biodiversity monitoring datasets in EDI with detailed Ecological Metadata Language (EML) and citable DOIs.

Search the EDI Data Portal or query the PASTA+ API to discover and download packages. The MBON Data catalog harvests EDI via the same PASTA search interface used by the `EDIutils` R package.
