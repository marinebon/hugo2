The Data Management working group (DMAC) supports every MBON node in meeting FAIR data principles. Activities include establishing Darwin Core / DwC-A submission workflows to OBIS, developing quality-control pipelines, maintaining the MBON data portal, and coordinating with IOOS DMAC and the Ocean Biodiversity Information System.

The group maintains shared code repositories, data submission templates, and training materials for new node teams.
